# Cryptocurrency Exchange KYC

## Overview

Crypto exchanges face unique KYC challenges — global user base, regulatory patchwork, high fraud rates, and resistance from privacy-focused users.

---

## Crypto KYC Challenges

| Challenge | Details |
|-----------|---------|
| **Global user base** | Users from 190+ countries, diverse documents |
| **Regulatory patchwork** | Different rules per jurisdiction (some ban crypto entirely) |
| **High fraud rate** | 5-10% fraud attempt rate (higher than banking) |
| **User resistance** | Crypto culture values privacy — friction causes churn |
| **Travel Rule** | FATF Travel Rule requires sender/receiver identification for transfers |

## Tiered KYC Approach

| Tier | Limits | Verification |
|------|--------|-------------|
| **Basic** | Low withdrawal limits | Email + phone only |
| **Standard** | Medium limits | Document + selfie |
| **Advanced** | Full limits | Document + selfie + address proof + source of funds |

---

## Key Takeaways

!!! success "Summary"
    - **Tiered KYC** balances user acquisition with compliance — basic for onboarding, upgrade for higher limits
    - **FATF Travel Rule** is driving KYC adoption across the crypto industry
    - **EU AMLR 2024** subjects all CASPs to full bank-equivalent KYC
    - High document diversity (190+ countries) makes broad document coverage essential
