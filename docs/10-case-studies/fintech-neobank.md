# Fintech / Neobank Onboarding

## Overview

Fintechs and neobanks optimize eKYC for **speed and conversion** — mobile-first, minimal friction, API-integrated.

---

## Fintech vs Bank Approach

| Aspect | Traditional Bank | Fintech/Neobank |
|--------|-----------------|----------------|
| **Primary channel** | Branch + digital | Mobile-only |
| **Onboarding time** | Minutes to days | < 2 minutes |
| **Risk tolerance** | Conservative | Higher (smaller account values) |
| **Technology** | Vendor SDK | API-first, custom UI |
| **Fallback** | Branch visit | Chat support, simplified flow |

## Typical Fintech Stack

| Component | Common Choices |
|-----------|---------------|
| **eKYC** | Onfido, Sumsub, HyperVerge |
| **Screening** | ComplyAdvantage, Refinitiv |
| **Device intelligence** | SEON, Sardine |
| **Orchestration** | Alloy, Persona, custom |
| **Risk scoring** | Custom ML + vendor signals |

---

## Key Takeaways

!!! success "Summary"
    - Fintechs achieve **< 2 minute** onboarding with **85%+ completion rate**
    - **API-first integration** with custom UI for brand consistency
    - **Higher risk tolerance** than banks — lower thresholds, fewer manual reviews
    - **Orchestration platforms** (Alloy, Persona) enable multi-vendor flexibility
