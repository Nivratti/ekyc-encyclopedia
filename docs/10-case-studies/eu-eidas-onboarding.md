# EU: eIDAS Compliant Onboarding

## Overview

eKYC in the EU under eIDAS — NFC chip reading for high assurance, cross-border recognition, and preparing for EUDI Wallet integration.

---

## EU eKYC Assurance Levels

| Method | eIDAS Level | Use Case |
|--------|-----------|----------|
| **Document + selfie** | Substantial | Standard remote onboarding |
| **Document + selfie + NFC chip** | High | High-assurance accounts |
| **Video identification** | Substantial-High | Premium/complex onboarding |
| **EUDI Wallet credential** | High (future) | Post-2026 credential-based verification |

## Cross-Border Challenges

| Challenge | Details |
|-----------|---------|
| **27 different national IDs** | Each country has different ID card format |
| **Multiple languages** | OCR must handle 24 official EU languages |
| **NFC availability** | Not all national IDs have NFC chips yet |
| **Regulatory interpretation** | Member states interpret AMLD differently |

---

## Key Takeaways

!!! success "Summary"
    - **NFC + biometric** is the gold standard for EU eKYC — achieves "High" assurance
    - **EUDI Wallet (2026-2027)** will transform EU eKYC — credential-based verification
    - **Cross-border** document diversity makes EU eKYC technically challenging
    - **AMLR 2024** harmonizes rules — simplifies compliance for pan-EU providers
