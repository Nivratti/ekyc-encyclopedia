# ID Document Types Worldwide

## Definition

An overview of the identity documents used globally for eKYC — passports, national IDs, driving licenses, and residence permits — covering formats, security features, and regional variations.

---

## Document Categories

| Category | Examples | Key Features |
|----------|---------|-------------|
| **Passport** | Booklet, card format | ICAO 9303 standard, MRZ, increasingly NFC-chipped |
| **National ID** | Aadhaar, NIE, Carta d'identità | Country-specific format, most common for domestic eKYC |
| **Driving License** | US DL, EU DL, Indian DL | Varies enormously (US: 50+ state formats) |
| **Residence Permit** | EU residence card, green card | Proof of right to reside |
| **Voter ID** | India EPIC, various | Common in developing countries |

---

## Regional Highlights

| Region | Primary ID Document | Key Characteristic |
|--------|-------------------|-------------------|
| **India** | Aadhaar (1.4B issued) | 12-digit number, QR code, linked to biometrics |
| **EU** | National ID card (eID) | Standardizing under eIDAS, NFC chip |
| **USA** | State driving license / passport | No national ID — 50+ different DL formats |
| **Singapore** | NRIC / Singpass | Linked to MyInfo digital identity |
| **UAE** | Emirates ID | Smart card with biometric chip |
| **Africa** | National ID / voter ID | Rapidly digitalizing (Nigeria NIN, Kenya Huduma) |

---

## Standard Document Sizes

| Format | Dimensions | Used For |
|--------|-----------|----------|
| **ID-1 (CR80)** | 85.6 × 53.98 mm | Credit card-sized ID cards, driving licenses |
| **ID-2** | 105 × 74 mm | Some national IDs (Germany old format) |
| **ID-3** | 125 × 88 mm | Passports (data page) |
| **Custom** | Varies | Aadhaar letter (A4), some voter IDs |

---

## Key Takeaways

!!! success "Summary"
    - **6,000+ document types** globally with hundreds of format changes annually
    - **Passports** are most standardized (ICAO 9303); **driving licenses** are most variable
    - **USA has no national ID** — making document verification harder (50+ DL formats)
    - **India's Aadhaar** is the world's largest biometric ID (1.4B), with QR code for offline verification
    - eKYC providers must continuously update their document databases
