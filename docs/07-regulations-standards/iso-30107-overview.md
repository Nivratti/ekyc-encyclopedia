# ISO 30107 (PAD Testing Standard)

## Overview

Comprehensive coverage of this standard is in the [Biometrics section](../02-biometrics-face/iso-30107-pad-testing.md). This article provides the regulatory context.

ISO/IEC 30107 defines the international standard for **Presentation Attack Detection (PAD)** testing. Part 3 specifies the methodology used by iBeta and other test labs. Increasingly referenced by regulators (EU AI Act, eIDAS 2.0) as the benchmark for biometric security testing.
